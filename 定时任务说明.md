# 定时任务配置说明

## 📋 你需要做什么

### 方式一：使用 Launchd（推荐，macOS）

在终端执行以下命令：

```bash
# 1. 复制定时任务配置文件到 LaunchAgents 目录
cp /Users/baoying/Desktop/AI简报/com.aiBrief.plist ~/Library/LaunchAgents/

# 2. 加载定时任务
launchctl load ~/Library/LaunchAgents/com.aiBrief.plist

# 3. 启动定时任务
launchctl start com.aiBrief.dailyUpdate
```

### 方式二：使用 Cron（通用）

```bash
# 编辑 crontab
crontab -e

# 添加以下行（每天早上7点执行）
0 7 * * * /usr/bin/python3 /Users/baoying/Desktop/AI简报/daily-update.py >> /Users/baoying/Desktop/AI简报/logs/cron.log 2>&1
```

### 方式三：手动测试（立即执行一次）

```bash
cd /Users/baoying/Desktop/AI简报
python3 daily-update.py
```

## ⚙️ 定时任务功能

每天早上 **7:00** 自动执行：

1. ✅ 搜索最新资讯（需要配置搜索API）
2. ✅ 生成播客脚本
3. ✅ 生成播客音频
4. ✅ 更新HTML链接
5. ✅ 记录运行日志

## 🛠️ 管理定时任务

### 停止定时任务

```bash
# Launchd
launchctl unload ~/Library/LaunchAgents/com.aiBrief.plist

# Cron
crontab -e
# 删除对应的行
```

### 重新启动定时任务

```bash
# Launchd
launchctl unload ~/Library/LaunchAgents/com.aiBrief.plist
launchctl load ~/Library/LaunchAgents/com.aiBrief.plist

# Cron 会自动重新加载
```

### 查看下次执行时间

```bash
# Launchd 不直接显示，但可以查看日志
tail -20 /Users/baoying/Desktop/AI简报/logs/update.log
```

## 💡 注意事项

1. **首次运行建议手动测试**，确保脚本正常工作
2. **日志文件会自动创建**，无需手动创建
3. **如果修改了脚本**，需要重新加载定时任务
4. **电脑需要保持开机或唤醒**，定时任务才会执行

## 🔄 修改执行时间

编辑 `com.aiBrief.plist` 文件中的 `StartCalendarInterval` 部分：

```xml
<key>StartCalendarInterval</key>
<dict>
    <key>Hour</key>
    <integer>7</integer>   <!-- 改成你想要的小时 -->
    <key>Minute</key>
    <integer>0</integer>   <!-- 改成你想要的分钟 -->
</dict>
```

修改后重新加载：
```bash
launchctl unload ~/Library/LaunchAgents/com.aiBrief.plist
launchctl load ~/Library/LaunchAgents/com.aiBrief.plist
```
