## 🎨 自定义配置

### 修改语音风格
编辑 `podcast-service.py` 第30行：
```python
self.default_voice = self.voice_options['news']  # 默认：新闻播报风格

# 可选：
# self.default_voice = self.voice_options['female']  # 女声
# self.default_voice = self.voice_options['male']    # 男声
```

### 修改脚本长度
编辑 `podcast-service.py` 第125行：
```python
for news in section['news'][:3]:  # 改成 [:2] 或 [:5] 调整每板块新闻数量
```

## 💡 常见问题

### Q: 音频无法播放？
A: 确保使用HTTP服务器访问，直接打开HTML文件可能有跨域问题。

### Q: 如何更改语音？
A: 编辑 `podcast-service.py` 中的 `default_voice` 配置。

### Q: 可以离线使用吗？
A: 可以安装pyttsx3替代edge-tts：
```bash
pip install pyttsx3
```

### Q: 如何部署到生产环境？
A: 建议使用Nginx或Caddy等Web服务器，配合自动化脚本每日更新。
